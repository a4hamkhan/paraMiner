# paraMiner
``` 
 git clone https://github.com/pwnesec/paraMiner.git
 cd paraMiner
```
# Usage
```
usage: paraMiner.py [-h] --url URL [--verbose] [--json]

options:
  -h, --help  show this help message and exit
  --url URL   URL to crawl
  --verbose   Print parameter names
  --json      Output parameters in JSON format
```
# Example
```
Normal Usage  : python3 paraMiner.py --url https://google.com --verbose
Json Format   : python3 paraMiner.py --url https://google.com --json
```
